# food-delivery-customer-churn-prediction
# Smart Retention: Predicting Food Delivery Customer Churn

## Project Overview
This project predicts customer churn for a food delivery platform using machine learning techniques.

## Objectives
Identify customers likely to stop using the service so businesses can take retention actions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps Performed
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Model training using classification algorithms
5. Model evaluation using accuracy and ROC-AUC

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Outcome
The model predicts potential customer churn and provides insights to help improve customer retention strategies.
